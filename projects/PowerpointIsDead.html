PowerPoint Is Dead.

Idea = Prezi + LESS

PowerPoint is dead. It's cold, corporate and looks ancient. Recently, new tools have risen in popularity to challenge the status quo held by PowerPoint, Apple's Keynote and Google's Slides. The most popular of which, has to be Prezi.

However, what's frustrating for a web developer, using a web based presentation tool, is knowing the capabilities of the platform and not being able to use them. It should be easy to embed content, link to external pages, import custom fonts, even apply custom event handling on certain keypresses using JavaScript.

There exist some JavaScript plugins for doing similar things. Such as impress.js and reveal.js, however, I'm yet to find one that allows the user to code their styles in a variable based system like LESS or SASS then compiles into CSS as part of a visual editor (because coding everything can be tedious).

So there's a project idea, should I ever find the time. A visual presentation tool that facilitates styling, through LESS or SASS, and custom JavaScript. Probably as a browser extension or Electron app to allow for offline use. Really want to build something using Electron at some point.
